How to Set Tension on the Gibs

This guide is for the A axis. It is the same process for the X, Y, and Z axis gibs as well.

  1. Select "Maintenance"

  2. Turn the Key to mode because you will need to move the slides while tightening the gibs using the buttons .

  3. Select the icon at the bottom of the screen to access the torque monitor window.

You may select the axis and the bar graph will show its tension. The tension on each slide will be shown as Torque on each axisâ€™ drive motor. 100 on the bar graph means that the tension on the gib requires 100% of the torque capability for the drive motor for the selected axis. If the tension is below 75 the graph color will be green.

If the tension is closer to the design capacity of drive motor the graph color will be yellow.

A reading over 100 means that there is risk of overheating and perhaps burning up the drive

motor. If the gib has been tightened too much, the drive motor is in danger of overheating and

graph color will be red.

  1. Set the tension on each axis so the bar graph indicates approximately 75.

Start by adjusting the A axis. The procedure for this axis is different from the others.

  1. Loosen each lock nut and then slightly loosen the set screw

 

 

  1. Use the buttons to move the A axis slide in the (-) direction .

  2. Use the buttons to move the A axis slide in the (+) direction .

  3. Continue moving the A axis slide in the (+) direction and (-) direction over the full travel distance approximately 4 times.

  4. Use the buttons  to move the A axis slide to the center position of the A axis base. The center of the slide should be aligned with the center of the belt sprocket.

  1. Number each of the set screws beginning at the grinding wheel end of the slide. There are 9 set screws.

  1. Start with set screw 5. Hold the short end of the hex wrench and tighten the set screw until it just comes in contact with the gib.

  2. Hold the hex wrench in position and tighten the lock nut.

  1. Continue using this process on the set screws in the following order: 4, 6, 3, 7.

  2. Move the A axis slide to position set screws 1 and 2 over the lower casting as much as possible without reaching the hard stop. Tighten the set screws using the same process as above.

  3. Move the A axis slide to position set screws 8 and 9 over the lower casting as much as possible without reaching the hard stop. Tighten the set screws using the same process as above.

  4. Use the axis torque meter to check the tension on the slide. Move the A axis slide over its complete travel and monitor the torque meter. The reading should be between 30 and 50 and be consistent across the full range of travel. A torque reading below the range can cause inconsistency in the diameter of the dies being ground. A torque reading above the range can cause excess wear or a fault that will stop the grinder from running to avoid damage to the machine.

  1. Repeat the process as necessary to achieve the proper torque range.

  2. Note: After adjusting a gib screw, if it makes a snapping noise / feel when loosening, it is likely too tight.

  3. Once the process is complete, check the axis torque meter while grinding and adjust again if needed. The axis torque values achieved while in "Man" mode can vary from when in "Auto" mode. Generally you will see less torque in Auto mode.
