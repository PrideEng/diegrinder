Dressing the Chuck Grinding Wheel with the Single Point Diamond

Before grinding the work holding chuck or chuck jaws, the grit wheel used for grinding the chuck must be dressed or the ground chuck will become distorted. Use the following procedures to dress the grit wheel used for grinding the chuck.

Do not use handheld grinding wheel dressing tools.

  1. Turn the key to the Auto position.

  2. Select Dressing and Regrinding

  3. Select Chuck Wheel Dressing

  4. Fill in the data fields with the required information in the Chuck Wheel Dressing window.

Make sure the Control Power On Icon is on before proceeding to step 5.   

  1. Move the cursor to the icon. Depress left mouse button until the green icon turns gray.

  2. Select the icon (OP3) and verify that the program "Chuck_Wheel_Dress.npg" is selected.

  3. Depress the Cycle Start button
