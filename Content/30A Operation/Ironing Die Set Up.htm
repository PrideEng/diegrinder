Ironing Die Set Up

The following is the process for setting up ironing die programs:

  1. Select an Ironing Die (in good condition) to be the “Master Die”. This die needs to be clearly marked as “Master Die” and maintained for future use.

  2. Turn key to mode.

  3. Select and then select program for the die you have chosen to be the “Master Die”.

  4. Check all of the other die geometry input fields for accuracy.

If changes are made to the geometry below the target ID block, the die offset must be recalculated. The offset is to the left of the program page. Click on the appropriate button then click Calculate offset.
