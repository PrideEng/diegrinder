Ironing Die Grinding

The following is the process for running ironing die programs:

  1. Turn key to mode.

  2. Select the ironing die to be ground and measure the ID.

  1. Insert the ironing die into the chuck.

  2. Check all of the other die geometry input fields for accuracy.

  3. Move the cursor to the    icon. Depress left mouse button until the green icon turns gray.

If changes are made to the geometry below the target ID block, the die offset must be recalculated. The offset is to the left of the program page. Click on the appropriate button then click Calculate offset.

 

  1. Enter the target ID for the ironing die to be ground in the field. The Target ID equals the measured ID plus the amount of material to be removed.

    Example

    Die ID = 67.3250 mm (2.6506”)

    Material removal of 0.010 mm (0.0004”)

    67.3250 mm + 0.010mm = 67.3350 mm

    2.6506” + .0004” = 2.6510”

    67.3350 mm (2.6510”) would be known as your target ID

Set the axis stroke limits if they have not been set previously for the ironing die type to be ground.

  1. Turn key to mode.

  2. Select and then select program for the die you have chosen.

  3. Select the icon (OP3) and verify that the correct program is selected.

  4. Depress the Cycle Start button  once to make the first move.

  5. Visually inspect for proper positioning of the die grinding wheel.

  6. Depress the Cycle Start button  again to start the program.
