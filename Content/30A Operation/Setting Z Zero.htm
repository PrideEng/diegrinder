  2. Turn key to mode.

  3. Select and then select program for the die you have chosen to be the “Master Die”.

  4. Measure the ID (inner diameter) of the “Master Die”

  1. Turn key to mode.

  2. Select and then select program for the die you have chosen.

  3. Select the icon (OP3) and verify that the correct program is selected.

  4. Depress the Cycle Start button  once to make the first move.

  5. Visually inspect for proper positioning of the die grinding wheel.

  6. Depress the Cycle Start button  again to start the program.

  7. Remove the ironing die from the chuck once the program is complete.

  8. Measure the ID of the ironing die and compare to the Target ID. If the measured ID is greater than or less than the Target ID by more than 0.0002 inches (0.005 mm), then zero needs to be adjusted. Adjust zero by half of the difference between the measured ID and the Target ID. For example, if the measured ID was 0.0004 inches greater than the Target ID, then move the axis in the (-) direction 0.0002 inches and then reset zero. If the measured ID was less than the Target ID, then adjust the axis in the (+) direction and reset zero.

  9. Repeat the steps as necessary until the measured ID compared to Target ID is within tolerance.
