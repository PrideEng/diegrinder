Setting Z zero/Ironing Die Grind Process

The grinding wheel needs to be dressed prior to this process.

  1. With Machine in Auto Mode, select the Ironing die program matching the die to be ground

  2. Insert the master die into the chuck.

  3. Enter the ID of the master die into the “Target ID of Die” section.

  4.  

  6. Click program select on the screen.

  7. Move to NC screen and verify program.

  8. Press cycle start on the keypad once to initiate first move.

  9. Switch the Key to Off.

  10. Move A axis to position the grind wheel in the center of the die.

  11.  

Move Z axis in the positive (+) direction towards the Die.

Slowly increment Z in the positive direction while manually rotating the chuck until you see the spindle start to slightly move with the chuck. (Do Not Impact the Grind Wheel to the Die)

 

The Die and the grind wheel should slightly touch. You should be able to move (1 micron for metric)

(2-3 millionths for inch) away from the die freely rotating without interference. Rotate the grind wheel spindle to different positions while checking the chuck rotation to give a more accurate touch-Off.

 

Move Z axis (1 micron for metric) (2-3 millionths for inch) in the negative (-) direction away from the Die. The Die and chuck should freely move without touching each other. Again, check at several positions of the grind wheel spindle. You may need to move more away from the Die for the Die and Chuck to no longer be touching.

 

Switch Machine to Auto and set Z zero.

Close Zero set screens

Switch the machine to manual mode and move the spindle away from the Die to a safe position. Move Z axis minus (-) then A axis minus (-).

 

Remove the master die.

Die Grind

Measure the ID of the ironing die to be ground.

Insert the die to be ground into the chuck.

In the program screen check all geometry inputs for accuracy. If changes are made to the geometry below the target ID block, the die offset must be recalculated. The offset is to the left of the program page. Click on the appropriate button then click Calculate offset.

Enter the target ID into the “Target ID of Die” section. Target ID equals measured ID plus material to be removed.

Ex. Die ID of 67.3250mm

Material removal of 10 microns

67.3250 + .010 = 67.3350 (67.3350 would be known as your target ID)

Switch the Machine to Auto mode.

Click program select on the screen.

Press cycle start on the keypad once to initiate first move.

Visually inspect for proper location prior to cycle start.

Set A strokes on the program screen if not yet set.

  • Manually move A axis in until the back edge of the grind wheel is almost off the backside of the land

  • Set A In

  • Manually move A axis until the front edge of the grind wheel is almost off the front edge of the land

  • Set A Out

  • Move A axis to a safe minus position and rotate C axis to the exit angle degree. The NC screen will show the C axis angle.

  • Manually move A axis in until the front edge of the grind wheel is past the exit angle but not hitting the chuck.

  • Set A In Exit Angle

  • Manually move A axis in until the front edge of the grind wheel is almost off the front edge of the exit angle (closest to land)

  • Set A Out Exit Angle In

Close center doors.

Press cycle start again to run the program.

When the program is done, remove die from the chuck.

Measure the die ID

If the ID is greater or less than target ID by 2 microns or more adjust Z zero.

To adjust Z zero after initial run switch the machine to manual mode.

Move Z axis to 0.000

  • Adjust Z by half the difference between measured and targeted ID.

  • Reset Z zero

    Ex. Target=67.3350

               Measured=67.3390

    Measured 4 microns over targeted ID. Move Z axis 2 micron in the minus direction and reset Z zero.

    Note: If ground die measures over target move Z axis minus (-) from die. If ground die measures under target move Z axis plus (+) from die.

Repeat steps 15-28 until appropriate zero is achieved.
