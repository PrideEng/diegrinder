Dressing the Grinding Wheel

One of the first tasks to be performed when starting a new grinder is to dress the grinding wheel. Dressing the die grinding wheel is a manual hand wheel operation but first you must set it up. The grinding wheel should just stay in contact with the dressing wheel at both ends of its stroke.

  1. Select and install the grinding wheel to be used in the wheel head spindle.

  2. Turn key to   mode.

  3. Use the buttons to move the axis to 0°.

  4. Install wheel dressing fixture on chuck in the same manner that a die is put on the chuck. If the grinder has a magnetic chuck, energize chuck to 50% to allow for easy alignment of fixture and steps 4a. and 4b. If the grinder has a 6-Jaw chuck, skip to step 5.

    2. Energize magnetic chuck to full power.
